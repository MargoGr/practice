Проект посвящён проектированию информационной системы веб-платформы для публикации, поиска и оценки творческих работ пользователей. В рамках работы были разработаны функциональные модели бизнес-процессов, выполнено проектирование базы данных и построены диаграммы, описывающие структуру и взаимодействие компонентов системы.

Диаграммы проекта выполнены в Business Studio и Enterprise Architect.
